## Install at local
### Open Terminal, then type command:

`git clone https://github.com/saumeya/Watson_RN_NODEjs.git`

### Go to project folder :

`cd Watson_RN_NODEjs`

### Type following command :

`npm install`

### To run on Android :

`react-native run-android`

### To run on iOS :

`react-native run-ios`
